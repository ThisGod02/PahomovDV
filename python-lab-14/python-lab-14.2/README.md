# Lab 14.2 - MongoDB Demo

This folder contains a practical implementation for the MongoDB part of lab 14.

Files:
- `seed_data.js` - seed script for a real `mongosh` session
- `mongo_queries.js` - CRUD and aggregation examples for `mongosh`
- `mongo_demo.py` - local Python demo using `mongomock` by default
- `docker-compose.yml` - MongoDB and Mongo Express for manual verification

Quick start:

```bash
pip install -r requirements.txt
python mongo_demo.py
```
