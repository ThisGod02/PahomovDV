# Лабораторная работа №10: Современная веб-разработка

## Описание
Данная лабораторная работа посвящена изучению основ современной фронтенд-разработки с использованием React и Next.js. Работа разделена на две части: создание To-Do приложения и разработка сайта-портфолио.

## Структура проекта
```
python-lab-10/
├── todo-app/            # Часть 1: React + Vite + TypeScript
├── portfolio-site/      # Часть 2: Next.js + Tailwind CSS
├── README.md            # Данный файл
└── Отчёт.md             # Отчёт о выполнении работы
```

## Часть 1: Todo App
Приложение для управления списком задач.
- **Стек:** React, Vite, TypeScript, Tailwind CSS.
- **Запуск:**
  ```bash
  cd todo-app
  npm install
  npm run dev
  ```

## Часть 2: Portfolio Site
Сайт-портфолио с блогом и динамической маршрутизацией.
- **Стек:** Next.js, TypeScript, Tailwind CSS.
- **Запуск:**
  ```bash
  cd portfolio-site
  npm install
  npm run dev
  ```
