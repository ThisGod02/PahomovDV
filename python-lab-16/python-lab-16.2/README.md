# Lab 16.2 - Airtable CRM Pack

This folder contains a practical support pack for the Airtable lab:

- ready CSV files for import
- exported base schema in JSON
- local script that computes formulas and rollups offline
- optional Airtable API upload if credentials are provided

Run local demo:

```bash
pip install -r requirements.txt
python airtable_sync.py --offline-demo
```
