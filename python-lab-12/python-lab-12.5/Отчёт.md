## 1. Цель работы
Получить практические навыки создания прогрессивного веб-приложения (PWA) на основе новостного сайта. Изучить ключевые компоненты PWA: Web App Manifest для установки на устройство, Service Workers для кэширования и офлайн-доступа, а также Push API для уведомлений.

## 2. Структура проекта
Приложение "PWA Новости":
- `server/` — Backend на Node.js (Express), работа с NewsAPI и Push-уведомлениями.
- `public/` — Frontend (HTML/CSS/JS).
- `manifest.json` — Описание приложения для установки.
- `sw.js` — Service Worker для кэширования.

## 4. Ответы на вопросы (по аналогии)
- **Что такое Service Worker?** Это скрипт, работающий в фоновом режиме, который позволяет кэшировать ресурсы и обрабатывать пуш-уведомления.
- **Чем PWA лучше обычного сайта?** PWA можно установить на экран "Домой", оно работает офлайн и имеет доступ к нативным функциям через браузерные API.
