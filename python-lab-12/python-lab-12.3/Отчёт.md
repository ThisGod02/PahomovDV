- `lib/services/` — Сервисы для работы с БД (sqflite) и датчиками (Location, Pedometer).
- `lib/providers/` — Управление состоянием (Provider).
- `lib/widgets/` — Компоненты интерфейса.

## 4. Ответы на вопросы (по аналогии)
- **Почему во Flutter используется Dart?** Dart оптимизирован для клиентских приложений, поддерживает Hot Reload и компиляцию в нативный код (AOT/JIT).
- **Что такое Provider во Flutter?** Это обертка над InheritedWidget, которая позволяет удобно передавать состояние вниз по дереву виджетов.
