## 3. Реализация TODO-задач

### 3.1. Работа с базой данных (database.ts)
Реализованы функции `addNote`, `deleteNote` и `updateNote` для работы с SQLite.

```typescript
export const addNote = (note: GeoNote): Promise<void> => {
    return new Promise((resolve, reject) => {
        db.transaction(tx => {
            tx.executeSql(
                'INSERT INTO notes (id, title, content, latitude, longitude, address, photoUri, createdAt) VALUES (?, ?, ?, ?, ?, ?, ?, ?)',
                [note.id, note.title, note.content, note.latitude, note.longitude, note.address || null, note.photoUri || null, note.createdAt],
                () => resolve(),
                (_, error) => { reject(error); return false; }
            );
        });
    });
};
```

### 3.2. Управление состоянием (notesSlice.ts)

export const saveNote = createAsyncThunk(

## 4. Ответы на вопросы (по аналогии)
- **В чем отличие React Native от Flutter?** React Native использует нативные компоненты через "мост" (или JSI), а Flutter отрисовывает UI самостоятельно через Skia/Impeller.
- **Зачем нужен Redux в мобильном приложении?** Для централизованного хранения данных, которые должны быть доступны на разных экранах (например, список заметок).
